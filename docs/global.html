Options

CSS Class Options

pulldown
Style panels as a pulldown menu.
upward
Place panels above the bar.
overflow-scroll
Show scroll bar when the window height is lesser than a panel height.
with-popup-hint
Show small triangles as popup hints. (Simply as a style.)
